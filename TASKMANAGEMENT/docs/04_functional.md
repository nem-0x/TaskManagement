# 5. 機能要件

### 5.1 カラム管理
- 初期カラムとして「Todo」「In Progress」「Done」の3列を表示する
- カラムの追加ができる（タイトルを入力して作成）
- カラムの削除ができる（配下のカードも同時に削除）

### 5.2 カード管理
- カードの追加（タイトルを入力して作成）
- カードの削除
- カードの編集（タイトルの変更）

### 5.3 ドラッグ&ドロップ
- カードをカラム間でドラッグ&ドロップして移動できる
- カラム内での並び替えができる

### 5.4 データ保存
- カラム・カードの状態はバックエンドDBに保存する
- ページをリロードしてもデータが保持される
