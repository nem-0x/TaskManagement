## 1. プロジェクト概要

### 1.1 背景・目的

チームや個人のタスクを視覚的に管理するため、Trello 風のカンバンボードアプリを開発する。
カード（タスク）をカラム間でドラッグ&ドロップして進捗状況を直感的に把握できることを目的とする。

### 1.2 対象ユーザー

- タスクを視覚的に整理・管理したい個人ユーザー

### 1.3 スコープ

**対象範囲:**
- カラム（ステータス列）の管理
- カード（タスク）の管理
- カード間のドラッグ&ドロップ移動
- データのバックエンドDB永続化

**対象外:**
- ユーザー認証・ログイン機能
- 複数ユーザーによる共同編集
- 通知機能
- ファイル添付
- コメント機能

## 2. 用語定義

| 用語 | 定義 |
|------|------|
| カラム | タスクのステータスを表す列（例：Todo、In Progress、Done） |
| カード | 個々のタスクを表す要素。カラムに属する |
| ボード | カラムとカードを配置するメインの作業画面 |
