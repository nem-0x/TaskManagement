# 7. 画面設計

### 7.1 画面一覧

| 画面ID | 画面名 | 説明 |
|--------|--------|------|
| SCR-01 | ボード画面 | アプリのメイン画面。カラムとカードを表示・操作する |

> ログイン機能がないため、画面は1画面のみ。

### 7.2 画面遷移図

```
[アプリ起動] → [ボード画面 (SCR-01)]
```

### 7.3 ボード画面（SCR-01）ワイヤーフレーム

```
┌──────────────────────────────────────────────────────────────┐
│  タスク管理アプリ                          [+ カラム追加]      │
├─────────────────┬──────────────────┬────────────────────────┤
│  Todo      [×]  │  In Progress [×] │  Done            [×]   │
│─────────────────│──────────────────│────────────────────────│
│ ┌─────────────┐ │ ┌──────────────┐ │ ┌────────────────────┐ │
│ │ タスクA  [×]│ │ │ タスクC   [×]│ │ │ タスクE        [×] │ │
│ └─────────────┘ │ └──────────────┘ │ └────────────────────┘ │
│ ┌─────────────┐ │                  │                        │
│ │ タスクB  [×]│ │                  │                        │
│ └─────────────┘ │                  │                        │
│                 │                  │                        │
│ [+ カード追加]  │ [+ カード追加]   │ [+ カード追加]         │
└─────────────────┴──────────────────┴────────────────────────┘
```

**UI要素説明:**

| 要素 | 説明 |
|------|------|
| `[+ カラム追加]` | クリックでカラム名入力フィールドを表示 |
| カラムタイトル横 `[×]` | カラム削除ボタン。クリックで確認ダイアログを表示 |
| カード横 `[×]` | カード削除ボタン。クリックで即時削除 |
| カードタイトル | クリックでインライン編集モードに切り替わる |
| `[+ カード追加]` | クリックでカード名入力フィールドを表示 |

**カラム追加時の入力フォーム:**

```
┌─────────────────────────────────┐
│  新しいカラム名を入力...          │
│  [追加]  [キャンセル]            │
└─────────────────────────────────┘
```

**カード追加時の入力フォーム:**

```
┌─────────────────┐
│ タスク名を入力...│
│ [追加] [×]      │
└─────────────────┘
```
