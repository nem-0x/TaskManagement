# 7. 画面設計

### 7.1 画面一覧

| 画面ID | 画面名 | 説明 |
|--------|--------|------|
| SCR-01 | ボード画面 | アプリのメイン画面。カラムとカードを表示・操作する |

> ログイン機能がないため、画面は1画面のみ。

### 7.2 画面遷移図

```
[アプリ起動] → [ボード画面 (SCR-01)]
```

**UI要素説明:**

| 要素 | 説明 |
|------|------|
| `[+ カラム追加]` | クリックでカラム名入力フィールドを表示 |
| カラムタイトル横 `[×]` | カラム削除ボタン。クリックで確認ダイアログを表示 |
| `[↑ 優先度順]` | クリックでそのカラムのカードを優先度順（高→中→低→未設定）に並び替え |
| `[↑ 期限順]` | クリックでそのカラムのカードを期限日の近い順に並び替え（未設定は末尾） |
| 優先度バッジ `[高][中][低]` | カードの優先度を色付きバッジで表示（高:赤、中:橙、低:緑） |
| 期限日 `📅 YYYY/MM/DD` | カードの期限日を表示。期限切れは赤字、残り3日以内は橙色で警告表示 |
| カード横 `[×]` | カード削除ボタン。クリックで即時削除 |
| カード本体 | クリックで編集モーダルダイアログを開く |
| `[+ カード追加]` | クリックでカード追加フォームを表示 |

**カラム追加時の入力フォーム:**

```
┌─────────────────────────────────┐
│  新しいカラム名を入力...          │
│  [追加]  [キャンセル]            │
└─────────────────────────────────┘
```

**カード追加時の入力フォーム:**

```
┌──────────────────────────┐
│ タスク名を入力...          │
│ 優先度: [なし ▼]          │
│ 期限日: [日付を選択]       │
│ [追加]  [キャンセル]       │
└──────────────────────────┘
```

```
├──────────────────────────────┤

└──────────────────────────────┘
```
