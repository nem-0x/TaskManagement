# 要件定義書 - タスク管理アプリ

## 1. プロジェクト概要

Trello 風のシンプルなタスク管理アプリ。ブラウザ上で動作し、データはローカルストレージに保存する。

---

## 2. 技術スタック

| 項目 | 内容 |
|------|------|
| フレームワーク | React + Vite |
| 言語 | TypeScript |
| スタイリング | Tailwind CSS |
| ドラッグ&ドロップ | @dnd-kit |
| データ保存 | localStorage |

---

## 3. 機能要件

### 3.1 カラム管理
- 初期カラムとして「Todo」「In Progress」「Done」の3列を表示する
- カラムの追加ができる（タイトルを入力して作成）
- カラムの削除ができる

### 3.2 カード管理
- カードの追加（タイトルを入力して作成）
- カードの削除
- カードの編集（タイトルの変更）

### 3.3 ドラッグ&ドロップ
- カードをカラム間でドラッグ&ドロップして移動できる

### 3.4 データ保存
- カードの状態はローカルストレージに保存する
- ブラウザを閉じて再度開いてもデータが保持される

---

## 4. 非機能要件

- シングルページアプリケーション（SPA）
- レスポンシブ対応は不要（デスクトップのみ）
- バックエンド・データベース不要

---

---

## 6. 開発ステップ

1. Vite で React + TypeScript プロジェクト作成
2. Tailwind CSS の導入
3. カラム・カード UI の実装
4. @dnd-kit によるドラッグ&ドロップ実装
5. localStorage による永続化
