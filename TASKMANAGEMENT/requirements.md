## 概要

Trello 風のカンバンボードアプリ。カード（タスク）をカラム間でドラッグ&ドロップして進捗を視覚的に管理する。対象ユーザーは個人。認証・共同編集・通知は対象外。

---

## セクション一覧

### [1. プロジェクト概要・用語定義](docs/01_overview.md)

背景・目的、対象ユーザー、スコープ（対象範囲／対象外）、用語定義（カラム・カード・ボード）を記載。

### [2. 技術スタック](docs/02_tech_stack.md)

フロントエンド: TypeScript + React 18 / Vite / Tailwind CSS / @dnd-kit / Zustand / TanStack Query。バックエンド: Java 21 + Spring Boot 3.x / Spring Data JPA / Flyway。DB: PostgreSQL。インフラ: Docker + Docker Compose。

### [3. ユースケース](docs/03_usecases.md)

6つのユースケース（カラム追加・削除、カード追加・編集・削除・移動）の基本フロー・代替フロー・事前／事後条件を記載。

### [4. 機能要件](docs/04_functional.md)

### [5. 非機能要件](docs/05_nonfunctional.md)

SPA構成、Chrome 最新版対応、1秒以内のレスポンス、HTTPS必須、エラーハンドリング、Git管理など。
