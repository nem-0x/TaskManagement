# 5. 機能要件

### 5.1 カラム管理
- 初期カラムとして「Todo」「In Progress」「Done」の3列を表示する
- カラムの追加ができる（タイトルを入力して作成）
- カラムの削除ができる（配下のカードも同時に削除）

### 5.2 カード管理
- カードの追加（タイトル・優先度・期限日を入力して作成）
- カードの削除
- カードの編集（タイトル・優先度・期限日の変更。モーダルダイアログで実施）
- 優先度は「高」「中」「低」の3段階（DB保存値: `high` / `medium` / `low`）。未設定も可
- 期限日は任意設定。期限切れ・期限間近（3日以内）は視覚的に警告表示する

### 5.3 ドラッグ&ドロップ
- カードをカラム間でドラッグ&ドロップして移動できる
- カラム内での自由な並び替えができる

### 5.4 並び替え
- カラム単位でワンクリック並び替えができる
  - **優先度順:** 高→中→低→未設定の順に並び替え
  - **期限順:** 期限日の近い順（昇順）に並び替え。期限未設定は末尾

### 5.5 データ保存
- カラム・カードの状態はバックエンドDBに保存する
- ページをリロードしてもデータが保持される
